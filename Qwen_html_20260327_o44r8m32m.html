Installation

  1. Install Satisfactory Mod Loader (SML) ^3.11.1
  2. Install dependencies: KLib ^4.3.0, ModularUI 2.1.35
  3. Install Refined Power ^4.0.43, FicsitFarming 4.2.34
  4. Install Satisfactory Plus via SMR or ficsit.app
  5. Start NEW save (incompatible with vanilla saves)

FAQ

Where is Caterium?

Caterium nodes don't exist in SF+. Find Aurovite and analyze it in MAM to unlock Caterium-related content [[9]]

Multiplayer support?

Multiplayer is NOT supported in Alpha phase. Attempting multiplayer will cause crashes [[9]]

Compatible with vanilla saves?

No! This mod overhauls the entire game. All new games must play through the tutorial or the game will crash [[20]]

How to unlock Ficsit Farming?

Explore the world and locate Dirt. Analyzing Dirt in the MAM unlocks the Ficsit Farming research tree [[9]]
